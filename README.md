# Cordova ExitIOSApp Plugin


## Using

Create a new Cordova Project

    $ cordova create hello com.example.helloapp Hello
    
Install the plugin

    $ cd hello
    $ cordova plugin add https://github.com/don/cordova-plugin-exit-ios.git


Install iOS platform

    cordova platform add ios
    
Run the code

    cordova run
